Integration Formulas

d/dx [x ] = 1

∫ 1 dx = x + C 

d/dx [ax] = a

 ∫ a dx = ax+ C

d/dx [xn] = nxn-1

∫ xn dx = ((xn+1)/(n+1))+C ; n≠1 

 d/dx [sin x] = cos x

∫ sin x dx = – cos x + C 

d/dx [cos x] = - sin x

∫ cos x dx = sin x + C 

d/dx [tan x] = sec2 x

∫ sec2 dx = tan x + C 

d/dx [cot x] = - csc2 x

 ∫ csc2 dx = -cot x + C

d/dx [sec x] = sec x . tan x

 ∫ sec x (tan x) dx = sec x + C

d/dx [csc x] = - csc x . cot x

 ∫ csc x ( cot x) dx = – csc x + C

d/dx [ln |x|] =  1/x

 ∫ (1/x) dx = ln |x| + C

d/dx [ex] = ex

 ∫ ex dx = ex+ C

d/dx [ax] = ax . ln |a|

 ∫ ax dx = (ax/ln a) + C ; a>0,  a≠1
